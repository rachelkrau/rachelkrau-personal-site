The Mentor Bank

Created a responsive site for mentors and mentees to create profiles and establish a connection based on experience level and common interests.

User Research, User Interviews, Personas, Journey Mapping, Design Sessions, Wireframes, Mockups, Prototyping, Usability Testing, Front-end Development

Vision Boards

Prior to starting development, vision boards were created to help narrow down a look and feel for the application.

Sketches & Design Sessions

I facilitated design sessions with the development team to get everyone involved with the UX process. These sketches led to high-fidelity designs.

HTML Emails

The application required communication with users via email. HTML emails were designed and written to match the look and feel of the site.

"Final" UI Design

Users had to access the application from multiple devices, including phones, tablets, and desktop computers. Our site had to accommodate all screen sizes.

Lessons Learned & Key Takeaways

  • There were many different user roles, but a lot of their goals and overall frustrations overlapped.
  • We tried out a new CSS framework, Skeleton, and had to write a lot of custom media queries to support smaller screen sizes.
  • Our stakeholders responded very well to off-the-cuff design sessions and wanted to be included in the process.
